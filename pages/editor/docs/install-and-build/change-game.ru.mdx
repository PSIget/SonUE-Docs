# Смена игры в Unreal Engine

## Подготовка

Для начала убедитесь, что у вас не загружен уровень со спавном, иначе это может привезти к вылету. Также желательно создать пустой уровень.

![New Level](./change-game.ru/img/new-level.png)

## Смена игры и перезагрузка XrGame
Выбираем нужную игру и ждём пока перезагрузится нужный XrGame

![Select Game](./change-game.ru/img/select-game.png)

**Примечание**
* Спавн автоматически обновляется после смены игры.
