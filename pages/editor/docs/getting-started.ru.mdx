# Начало работы

Добро пожаловать в документацию по проекту STALKER on UE! Это руководство поможет вам начать работу с проектом и предоставит необходимую информацию для начала вашего путешествия в мир STALKER on UE.

## Установка

Для начала работы с STALKER on UE вам необходимо выполнить следующие шаги установки:

1. Скачайте и установите последнюю версию Visual Studio
    - В параметрах установки выберите `Разработка игр на C++`
2. Скачайте и установите необходимую версию Unreal Engine (см. [Git](https://git.s2ue.org/RedProjects/SonUE))
3. Изучите документацию из раздела [Установка и сборка](./install-and-build/build)

## Системные требования

Перед тем, как погрузиться в разаботку STALKER on UE, убедитесь, что ваша система соответствует минимальным требованиям:

- **Операционная система:** Windows 10 20H2 64-bit и новее
- **Процессор:** Intel® Core™ i5-9400F / AMD Ryzen™ 5 2400G
- **Память:** 16 GB RAM
- **Графика:** NVIDIA GeForce RTX 3060 / AMD Radeon RX 6600 XT
- **Хранилище:** 256 GB SSD
