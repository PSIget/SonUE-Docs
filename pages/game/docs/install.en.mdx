# Installation

Welcome to the STALKER on UE installation section. Please follow the steps below to properly install the modification and start playing.

## Prerequisites

1. A copy of the original game installed on your computer.

<Callout type="info">
  It is required only to verify that you own the game. The modification will be installed in any folder that is convenient for you.
  Unfortunately, this is necessary to comply with copyright law.
</Callout>

2. Your system meets the system requirements.

## System Requirements

<Callout type="warning">
  Processor AVX instructions support is required!
</Callout>

Before diving into the development of STALKER on UE, make sure your system meets the minimum requirements:

- **Operating System:** Windows 10 20H2 64-bit or newer
- **Processor:** Intel® Core™ i5-9400F / AMD Ryzen™ 5 2400G
- **Memory:** 16 GB RAM
- **Graphics:** NVIDIA GeForce RTX 3060 / AMD Radeon RX 6600 XT
- **Storage:** 8 GB SSD

## Installation Steps

<Callout type="warning">
  Please note that the game installation path should not include special characters or letters other than those in the English alphabet, including Cyrillic.
</Callout>

This section will help you download, install, and launch the modification.

<Steps>
### Downloading the Installation Wizard

Click on the "Download Free" button on the [main page](https://s2ue.org/) of the website to download the installation wizard.

### Launching the Installation Wizard

Run the `setup.exe` file in the downloaded folder.

### Installation

Please follow the instructions of the installation wizard.

### Launching the Modification

Launch the modification from the shortcut on your desktop.

</Steps>

## Potential Problems and Their Solutions

- `Assertion failed: RenderBatch.NumIndices > 0`. This problem occurs for a limited number of people and has not yet been resolved by us.

We hope that the installation will go smoothly, and you will enjoy playing with the new mod!
