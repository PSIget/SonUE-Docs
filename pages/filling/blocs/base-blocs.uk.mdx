# Основні блоки

## Кроки (Steps)

Обов'язково використовуйте заголовок третього рівня `### Заголовок` для нумерації

```mdx
import { Steps } from 'nextra-theme-docs'

<Steps>
### Крок 1

Текст

### Крок 2

Текст

</Steps>
```

Результат буде наступним:

<Steps>
### Крок 1

Lorem Ipsum є просто вказівкою друкарського та верстатного промислу. Lorem Ipsum є стандартною вказівкою галузі ще з XVI століття, коли невідомий друкар взяв набір тексту та перемішав його, щоб зробити книгу зразкових типів.
