# Початок роботи

## Організація файлів

Спочатку система збирає всі ваші файли Markdown і конфігурації з каталогу `pages`, а потім генерує "інформацію про карту сторінок" вашого веб-сайту для відображення елементів, таких як _навігаційна панель_ і _бічна панель_ нижче:

### Структура файлів

Структура файлів має такий формат: `{категорія}/{ім'я-файлу}.{мова}.mdx`.

Доступні мови:

- `ru` - Російська
- `en` - Англійська
- `uk` - Українська

## Стандартна поведінка

За замовчуванням карта сторінок містить всі імена файлів `.md` та `.mdx` і структуру каталогів, відсортовані за алфавітом. Потім система використовуватиме пакет [title](https://github.com/vercel/title), щоб отримати форматовані назви сторінок з назв файлів.

Наприклад, якщо у вас є наступна структура:

```text
pages/
  contact.md
  index.mdx
  about/
    legal.md
    index.mdx
```

Отримана карта сторінок виглядатиме наступним чином (зверніть увагу, що всі назви відсортовані за алфавітом):

```json
[
  {
    "name": "About",
    "children": [{ "name": "Index" }, { "name": "Legal" }]
  },
  { "name": "Contact" },
  { "name": "Index" }
]
```

А глобальна карта сторінок буде об'єднана з кожною сторінкою документації. Потім налаштована тема відображатиме фактичний користувацький інтерфейс за допомогою цієї карти сторінок.

## Конфігурація `_meta.json`

Дуже часто потрібно налаштувати заголовок кожної сторінки, а не просто використовувати ім'я файлу. Наприклад, мати сторінку з заголовком "Index" не має сенсу, замість цього краще використовувати заголовок "Home".

Ось де на допомогу приходить файл `_meta.json`. Ви можете помістити файл `_meta.json` в кожен каталог, і він буде використовуватися для перевизначення конфігурації за замовчуванням для кожної сторінки:

```text
pages/
  _meta.json
  contact.md
  index.mdx
  about/
    _meta.json
    legal.md
    index.mdx
```

І ви можете помістити наступний код у файл `pages/_meta.json`:

```json filename="pages/_meta.json"
{
  "index": "Моя домашня сторінка",
  "contact": "Зв'яжіться з нами",
  "about": "Про нас"
}
```

Він повідомляє системі порядок кожної сторінки та правильний заголовок. Крім того, ви можете зробити це за допомогою `title` та додати до нього інші конфігурації:

```json filename="pages/_meta.json"
{
  "index": "Моя домашня сторінка",
  "contact": "Зв'яжіться з нами",
  "about": {
    "title": "Про нас",
    "...додаткові конфігурації...": "..."
  }
}
```

Додаткові конфігурації передаються в **тему** як додаткова інформація. Для отримання додаткової інформації ознайомтесь з відповідними сторінками.
