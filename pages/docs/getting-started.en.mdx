# Getting Started

Welcome to the documentation for the STALKER2UE project! This guide will help you get started with the project and provide the necessary information to begin your journey into the world of STALKER2UE.

## Installation

To get started with STALKER2UE, you need to follow these installation steps:

1. Download and install the latest version of Visual Studio
    - Select "Game Development with C++" in the installation options
2. Download and install the required version of Unreal Engine (see [GitHub](https://github.com/RedPandaProjects/Stalker2UE))
3. Review the documentation in the "Installation and Build" section (./install-and-build/build)

## System Requirements

Before diving into STALKER2UE development, make sure your system meets the minimum requirements:

- **Operating System:** Windows 10 20H2 64-bit or newer
- **Processor:** Intel® Core™ i5-9400F / AMD Ryzen™ 5 2400G
- **Memory:** 16 GB RAM
- **Graphics:** NVIDIA GeForce RTX 3060 / AMD Radeon RX 6600 XT
- **Storage:** 256 GB SSD
